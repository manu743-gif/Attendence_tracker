# 📊 Attendance Tracker Web App

A simple and efficient **web-based attendance tracker** built using **HTML, CSS, and JavaScript**, with **Firebase integration** for real-time data storage.

---

## 🚀 Features

* ✅ Mark daily attendance easily
* 📅 Track attendance history
* 🔄 Real-time database using Firebase
* 💻 Responsive design (works on mobile & desktop)
* ⚡ Fast and lightweight (no frameworks)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend / Database:** Firebase Realtime Database
* **Hosting:** Firebase Hosting (optional)

---

## 📂 Project Structure

```
Attendance_tracker/
│
├── index.html       # Main UI
├── style.css        # Styling
├── script.js        # Logic & Firebase integration
├── firebase.json    # Firebase config
├── .firebaserc      # Firebase project settings
└── .firebase/       # Firebase cache files
```

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the Repository

```
git clone https://github.com/manu743-gif/Attendence_tracker.git
cd Attendence_tracker
```

---

### 2️⃣ Firebase Setup

1. Go to Firebase Console
2. Create a new project
3. Enable **Realtime Database**
4. Copy your Firebase config
5. Paste it inside `script.js`

---

### 3️⃣ Run the Project

Just open:

```
index.html
```

OR use Live Server in VS Code.

---

## 🌐 Deployment (Optional)

To deploy using Firebase:

```
firebase login
firebase init
firebase deploy
```
